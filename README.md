# boilerplate-microservice-spring
Boilerplate design for microservices architecture with Spring

### Project Stack:

- ☕ Java 21
- 🌱 Spring 3.3.4
- ⚡ Hibernate
- 📚 Lombok
- 📖 OpenAPI
- 🧪 JUnit
- 🏛️ JPA
- 🚀 Gradle
- Spotless
- Pre-Commit
- Conventional Commits
- Reckon to Auto Versioning
- Aspect to Log in and out of applications methods
